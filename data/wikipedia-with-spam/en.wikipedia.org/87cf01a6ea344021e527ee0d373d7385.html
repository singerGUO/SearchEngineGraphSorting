Android Things

From Wikipedia, the free encyclopedia
Jump to: navigation, search

Android Things (codenamed Brillo) is an Android-based embedded operating system platform by Google, announced at Google I/O 2015. It is aimed to be used with low-power and memory constrained Internet of Things (IoT) devices, which are usually built from different MCU platforms.[1][2] As an IoT OS it is designed to work as low as 32–64 MB of RAM.[3] It will support Bluetooth Low Energy and Wi-Fi.[4] Along with Brillo, Google also introduced the Weave protocol, which these devices will use to communicate with other devices and which it hopes will be adopted by other IoT operating systems.[5] Every Android device recognize automatically any Brillo OS or Weave API based device. Users can choose a device, set it up and use it immediately.[6]
